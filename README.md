# Cubic Fruits
### A Minecraft Java mod that aims to add items from Blox Fruits and (sometimes) One Piece (as I watch it). 

Made with [MCreator](https://mcreator.net/)

**This is just the mod's code; you can get the mod itself here:**

- [CurseForge](https://www.curseforge.com/minecraft/mc-mods/cubic-fruits)

- Modrith might be coming soon after the release of the mod's remake.
